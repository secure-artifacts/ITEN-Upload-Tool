🔥 Firebase 实时数据库

配置团队的 Firebase 项目以启用打卡和实时同步功能。

📖 如何获取 Firebase 配置？
  1. 打开 Firebase Console
  2. 点击「添加项目」，输入项目名称，按提示完成创建
  3. 进入项目后，点击左侧齿轮 ⚙️ →「项目设置」
  4. 在「常规」标签页底部的「您的应用」区域，点击 Web 图标 </> 添加 Web 应用
  5. 注册应用后，会显示 firebaseConfig 对象，从中复制：
    • apiKey → API Key
    • authDomain → Auth Domain
    • databaseURL → Database URL（需先启用 Realtime Database）
    • projectId → Project ID
  6. 在左侧菜单 →「构建」→「Realtime Database」→「创建数据库」（选择区域后启动）
  7. 在左侧菜单 →「构建」→「Authentication」→「开始」→ 启用「Google」登录方式

使用说明

1. 获取 Gemini API Key，填入上方，点击"测试"验证。

2. 在词库文本框填入你希望 AI 选择的关键词。

3. 在"命名规则"页中，将 AI关键词 token 拖到你想要的位置。

4. 上传文件时 AI 会自动分析每张图片，从词库中选取最匹配的关键词填入文件名。

AI 命名会在上传前分析图片，每张图片约需 1-3 秒。如果命名模板中没有 {{aiKeywords}} token，AI 分析结果不会影响文件名。
